Sistema de Gestión de Biblioteca

Este proyecto implementa una jerarquía de materiales (Libros, Revistas, DVDs) utilizando **Java**. 

Características
- Clases Abstractas:** Uso de `MaterialBiblioteca` como base.
- Polimorfismo:** Cálculo dinámico de multas según el tipo de material.
- Despacho Dinámico:** Gestión uniforme de diferentes objetos en una sola lista.

Ejecución
Para correr el proyecto, utiliza la clase `Main` ubicada en el paquete `app`.
